### 计算机爱好者和计算机专业生的差别在哪里？

```
这是一篇多年前刚入行时自己写下的最满意的文章，时过境迁，此时此刻回首再看，发现早已物是人非。
```

#### 引言

```
在大多数人的观点里，一个计算机专业的人和一个非专业出身的人差别应该就是这4年积累的专业知识，以及在大学中做项目的经验，确实，这个经验很重要，但是这个经验只能一点点去弥补，不可能一口气吃成一个胖子。但是有一个非常重要的东西，确可以让我们的学习有一个质的提升，它就是：哲学。计算机从哪里来？到哪里去？要做什么？  

不识庐山真面目，只缘身在此山中，专业生我不知道，但是大多数非专业的新人菜鸟我相信对于计算机行业的理解和我一样，很符合这一句话，我们接触这一行并没有扎实的基础，也没有面向对象的概念，就如井底之蛙，在井底看天，在山中看山，这么一来我们的视野就会非常的狭小，所以大多数人都不知道自己在做什么。

这涉及到一个概念，OO，面向对象。在计算机的世界中，万物皆对象，那么可以借用这句话说，计算机中，万物皆程序，万物皆语言。
```

#### 万物皆对象

```
你可以把计算机当成一个对象，而在计算机中有无数的对象，比如硬件对象，软件对象，操作系统对象，浏览器对象，各端对象等，而面向对象最重要的一点是黑匣子，抛开具体的作用可复用性等，只看它的虚拟概念黑匣子，什么黑匣子，黑匣子就是一个封闭的对象，所以你可以不理解这些黑匣子，你只需要会使用黑匣子暴露出来的这些接口就可以了，对于所有人，比如使用手机你并不需要这些软件怎么做出来的，只需要怎么去操作，比如玩游戏，你不需要游戏怎么做出来的，你只需要学习别人设定好的操作。
```

#### 万物皆程序

```
计算机里面这些对象都是什么？其实说白了全部都是程序，系统内核，操作系统，电脑软件，手机软件，浏览器内核，浏览器插件，游戏。只是它们的功能以及实现它们的语言不一样，然而他们的最终目的都是为了方便人类的操作而诞生的。
```
    
#### 万物皆语言

```
那么说到程序，这些程序怎么来的？当然是由各种不同的语言实现的。比如说计算机语言，汇编语言，高级程序语言，端语言(我乱起的一个名字,只是自己的理解)，这些语言是怎么构成这个盘大的计算机世界的？

先说两个概念，编译器，解析器。

编译器：类似于翻译。高级程序语言(c,c++,java等)和低级语言(计算机语言)之间的翻译。

引擎：类似于翻译+运行。处理端语言脚本的虚拟机。

首先，为了让人类更好的操作硬件，有了用汇编语言和高级程序语言写出来的程序，这种程序叫操作系统(os)，世界四大操作系统,maxos,windows,unix,linux，其中linux算是由unix发展来的吧。

后来，为了方便普通人使用和程序员的开发使用，   操作系统又分为图形界面和字符界面，所谓的字符界面，原始的dos操作界面，unix界面，再后来出现了linux， 各大厂商在linux核心上开发，所以由了centos, ubuntu,红帽，suse等，操作系统核心和平常我们嘴里的操作系统有什么区别？ 操作系统核心是基础，如果没有了操作系统核心，所有程序都将运行不了，而这些所谓的centos,ubuntu等操作系统只是各大厂商在这些操作系统核心上生成出来的shell外壳，也就是供开发人员使用的软件，开发人员在上面开发更高层的软件，用来面向普通人，比如说APP，网站，游戏等。

说到网站，就不得不说到浏览器，浏览器也是一种在操作系统上开发出来的软件，也就是说如果没有操作系统，那么浏览器就使用不了。和操作系统一样，世界上也有好几种流行的浏览器核心，例如Trident(IE)，Gecko内核(Firefox),WebKit内核(曾经的Chrome,Safari),Presto内核(Opera)，Blink内核(现在的Chrome)，IE的内核真的是很多程序员深通恶绝的浏览器内核，那么360浏览器，猎豹浏览器是啥？只是基于这5大浏览器内核的浏览器插件。

端语言又是什么？例如这两年流行的javascript，或者称为前端，所谓的javascript是 ECMAscript + DOM + BOM，其中DOM和BOM都是浏览器对于ECMAscript暴露出来的接口，所以说ECMAscript是一门浏览器语言，但是就像不同厂商操作系统暴露出的参数也不完全一样，所以各浏览器内核厂商对于ECMAscript语言暴露的接口更是大不相同，这就导致上面提到的程序员深通恶绝的IE浏览器兼容性问题，不过渐渐的有了改善，所以说，计算机的发展都是朝着便捷前进的，因为便捷可以带来利益。

所以说：对于语言开发人员来说，并不需要知道这门语言是怎么来的，只需要学习这门语言暴露出来的接口如何使用，那么我们绝大多数的计算机行业人员在做的事情都是什么？我们都在使用别人创造的黑盒子上造盒子。
```

#### 感想

```
为什么一个浏览器语言会那么流行？因为它让人和计算机的交互更加的多样化和便捷，以及Nodejs，Nodejs是什么？Nodejs是用C++写的一款，可以让所谓的浏览器语言向底层迈进一步的东西，它的功能太强大，不好定义，它的特点：由无数个模块构成，各模块之间的交互有特定的参数方法，这就很像一个操作系统了，要知道操作系统也是由无数个单一功能的小程序构成的，例如Linux中，这无数个单一功能的小程序就是指令，也就是bin目录下的二进制源码包。

然而计算机一直都在前行，人类已经不满足于前两年流行的浏览器，手机app之间的交互，随着技术的发展，人工智能的崛起，Python的火爆是必然的趋势。

那么说到底，计算机爱好者和计算机专业生的差别在哪里？我想最重要的，就是思想，应该从整体去看待一个事物，再去深入钻研它的某一个领域。
```
